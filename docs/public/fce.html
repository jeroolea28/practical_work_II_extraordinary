Fundamentals of Computer Engineering

Class Overview

This class provides a global vision of Computer Engineering from both academic and professional perspectives. It introduces the theoretical foundations of the field while also helping students develop practical skills, particularly in web development.

The subject is divided into two modules: Theoretical Foundations and Practical Skills.

Theoretical Foundations

  • Computer Engineering in Society: History, role of engineers, ethical and anthropological foundations.
  • Information and Data: Representation, data organization, number systems, codes, and data ethics.
  • Processing & Transmission: Systems, networks, current paradigms (Cloud, AI, IoT, Blockchain, etc.).

Practical Skills

  • Software Development Basics: IDEs, Git, GitHub.
  • HTML5: Page structure, DOM, forms.
  • CSS3: Layouts, styling, box model, element positioning.

The main project in this part involves building a personal website using HTML and CSS. (Website you're reading right now!)

Objectives

  • Understand the role of computer engineering in society and the value it brings.
  • Gain historical perspective on technology development.
  • Recognize the human and ethical side of engineering.
  • Get an overview of current and emerging computing paradigms.
  • Begin exploring your future role and possible career paths in the field.

Learning Outcomes

  • Use basic terminology and concepts in both hardware and software.
  • Relate historical events to computing evolution and analyze trends.
  • Understand data representation methods and information systems.
  • Identify and explain the structure and functions of computer systems.
  • Develop a basic website using HTML and CSS for information sharing.
  • Present a computer-related project in both written and oral form.
